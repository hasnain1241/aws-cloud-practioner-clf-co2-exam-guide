# aws-cloud-practioner-clf-co2-exam-guide
This is your one source for the exam guide for aws cloud practioner. It includes prep pdfs, exam questions and links to a few pdfs


My suggestions to anyone attempting this exam is that you at most need a week to prepare for this exam if you have any prior knowledge of these services, i will suggest taking the md files of the sample exams and putting them into claude and asking it to make a dynamic exam, one which works same as aws exam, this way you can prepare in real time. It is nor too easy nor too difficult this exam, its about right, memorize all services, remember the hints and yes it will work for you. Try to book the exam physically i.e. at a test centre, the online exam has some issuses (check on reddit for more info)

aws also allows you to review the answers once done, Good Luck to anyone attempting this exam.

Note: The pdfs attached are just a reference for you all, you may need to prepare or look at more services and aws can test you any service they want.
